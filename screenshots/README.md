# Screenshots for Nextcloud App Store

This directory contains screenshots for the Project Control app listing on the Nextcloud App Store.

## Required Screenshots

For the Nextcloud App Store, you need to provide the following screenshots:

### 1. Main Dashboard (dashboard.png)
- Shows the main dashboard with project overview
- Displays statistics, recent projects, and quick actions
- Recommended size: 1280x720px or larger

### 2. Projects List (projects.png)
- Shows the projects management interface
- Displays project cards with status, budget, and team information
- Recommended size: 1280x720px or larger

### 3. Project Detail (project-detail.png)
- Shows detailed project view with time entries, budget info, and team members
- Displays project statistics and progress
- Recommended size: 1280x720px or larger

### 4. Time Entries (time-entries.png)
- Shows time tracking interface with entries list
- Displays time entry creation form
- Recommended size: 1280x720px or larger

### 5. Customer Management (customers.png)
- Shows customer management interface
- Displays customer list and detail views
- Recommended size: 1280x720px or larger

### 6. Settings (settings.png)
- Shows admin and personal settings
- Displays configuration options
- Recommended size: 1280x720px or larger

## Screenshot Guidelines

- Use PNG format for best quality
- Minimum resolution: 1280x720px
- Maximum file size: 2MB per screenshot
- Show the app in a realistic usage scenario
- Include actual data (not placeholder text)
- Ensure good contrast and readability
- Use a clean, professional appearance
- Show the app in both light and dark themes if applicable

## How to Take Screenshots

1. Set up a test Nextcloud instance with sample data
2. Create realistic projects, customers, and time entries
3. Use browser developer tools to set viewport to 1280x720
4. Take screenshots of each major feature
5. Ensure all text is readable and UI elements are clear
6. Save as PNG files with descriptive names
