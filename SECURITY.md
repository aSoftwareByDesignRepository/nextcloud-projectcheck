# Security policy

**Please do not report security vulnerabilities through public GitHub issues.**

- **E-mail:** [info@software-by-design.de](mailto:info@software-by-design.de)  
- Include a short description, affected versions if known, and steps to reproduce.

We will acknowledge receipt as soon as practical and coordinate disclosure after a fix is available.

For general bugs and feature requests, use [GitHub Issues](https://github.com/aSoftwareByDesignRepository/nextcloud-projectcheck/issues) in this repository.
